- Photos -

Different types of images:

  • Illustrations
  • Product photos
  • Patterns
  • Storytelling photos

Useful rules:

  1. Use images to support your website’s message and story. So only use relevant images!
  2. Prefer original images. If not possible, use original-looking stock images (not generic ones!)
    3. link_1Unsplash link_1Pexels link_1DrawKit link_1unDraw
  3. Try to show real people to trigger user’s emotions
  4. If necessary, crop images to fit your message
  5. Experiment combining photos, illustrations and patterns
  6. Darker or brighten image (completely or partially, using a gradient)
  7. Position text into neutral image area
  8. Put text in a box
  9. To account for high-res screens, make image dimensions 2x as big as their displayed size
  10. 1 Compress images for a lower file size and better performance
  11. When using multiple images side-by-side, make sure they have the exact same dimensions
